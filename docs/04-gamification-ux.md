# 게이미피케이션 & UI 설계

## 1. 디자인 컨셉
- 무드: **RPG 퀘스트 로그** (Diablo / 젤다 메뉴 느낌의 카드 + 양피지 톤 + 픽셀 아이콘 한두 개).
- 톤 & 매너: 비서가 아니라 **NPC 길드장**. ("모험가여, 다음 임무는 회의실 A에서 기다리고 있다네.")
- 컬러 팔레트:
  - Main Quest: `#F59E0B` (호박색)
  - Sub Quest: `#3B82F6` (퀘스트 마커 블루)
  - Side Quest: `#10B981` (에메랄드)
  - Done: `#9CA3AF` (회색)

## 3. 핵심 컴포넌트 사양

### 3.1 `VoiceInput`
- 상태: `idle | listening | processing`.
- 인터랙션: 스페이스바 길게 누르기(Push-to-talk) 또는 클릭 토글.
- 시각 피드백: 듣는 중에는 마이크 주변 펄스 애니메이션.
- 인식된 텍스트는 ChatPanel 입력칸에 실시간 미리보기.

### 3.2 `ChatPanel`
- 메시지 타입: `npc`, `user`, `system`(레벨업 등).
- NPC 메시지 우측에 작은 픽셀 아바타.
- 마지막 NPC 메시지 하단에 **퀵 액션 버튼**: `완료`, `다음`, `건너뛰기`.

### 3.3 `CharacterCard`
- 캐릭터 아바타(SVG, 한 종류면 충분).
- Lv, XP 게이지(애니메이션), 칭호(예: "회의실의 수호자").
- 오늘 완료된 퀘스트 카운트 / 총 카운트.

### 3.4 `QuestBoard`
- 탭: `Main / Sub / Side / Done`.
- 카드 항목:
  - 좌측 색띠로 종류 구분.
  - 제목, 시간, 장소 칩, 카테고리 이모지.
  - 우측 작은 액션: ✅(완료) / ✖(건너뛰기).
- Active 퀘스트는 카드 외곽에 글로우 효과.

### 3.5 액티브 퀘스트 강조 영역(상단)
- 카드 한 장으로 "지금 할 일"만 크게.
- 잔여 시간 카운트다운(분 단위).

## 4. NPC 말투 가이드 (프롬프트에 주입)

```
[규칙]
- 사용자를 "모험가" 또는 "용사"로 부른다.
- 한국어, 1~2문장, 과장은 살짝만.
- 시간/장소가 있으면 반드시 언급.
- 끝에 콜투액션을 한 줄로.

[예시]
- "용사여, 11시까지 회의실 A로 향하라. 그 전에 보고서 두루마리를 완성해야 한다네."
- "헬스장에서의 단련, 잊지 말게. 09시 정각이 되면 출발이네."
- "퀘스트 클리어! +30 XP. 다음 임무가 기다리고 있다네."
```

## 5. XP & 레벨 곡선

| Quest Kind | XP |
|---|---|
| Side | 10 |
| Sub | 30 |
| Main | 100 |
| 시간 내 완료 보너스 | +20% |

레벨업 임계치:
```
xpForLevel(L) = 100 * L + 50 * L * (L-1)
// L1→L2: 100, L2→L3: 250, L3→L4: 450, ...
```

레벨업 시:
- 토스트 + 캐릭터 카드 깜빡임 + NPC 시스템 메시지("⬆ Lv.8 달성! 칭호 획득: 시간 마법사").

## 6. 칭호 시스템 (가벼운 재미요소)
- 카테고리별 누적 완료 수로 칭호 부여.

| 조건 | 칭호 |
|---|---|
| work 10회 | 회의실의 수호자 |
| health 10회 | 강철의 모험가 |
| study 10회 | 도서관 마법사 |
| errand 10회 | 심부름의 달인 |
| 하루 5개 클리어 | 일일 퀘스트 마스터 |

## 7. 사운드 (옵션, 시간 남으면)
- 퀘스트 클리어: 짧은 "딩"
- 레벨업: 8-bit 팡파레
- 비활성화 토글 제공.

## 8. 접근성
- 모든 버튼 ≥ 44×44 px.
- 색 대비 WCAG AA 준수.
- 음성 입력 실패 알림은 텍스트로도 동일하게 노출.
